# 📊 TradeWithSmart - Pro Grid Engine

Welcome to **TradeWithSmart**, a professional execution suite designed for crypto grid trading simulations, live charts, and historical market analysis.

## ✨ Key Features
* 🔄 **Live Grid Trading Simulation:** Simulate grid trading strategies in real-time with customizable capital and margin layouts.
* 📈 **Market Summary Metrics:** Track highest, lowest, and average close prices along with detailed data points at a glance.
* 🛠️ **Grid Range Matrix Configuration:** Easily set manual price limits (Lower & Upper) and customize the number of grids.
* 📅 **Historical Candle Price Log:** Fetch and analyze historical charts and raw data logs for various crypto pairs (e.g., XRP/USDT, BTC/USDT).

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Language:** Python 3.14+
* **Data Feed Platform:** Bitget API Integration
